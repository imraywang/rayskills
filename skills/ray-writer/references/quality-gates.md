# 四层质量闸门

每层都必须通过。不能用语言精彩抵消事实薄弱，也不能用事实很多掩盖文章难读。

## 第一层 真实性

- 核心事实是否能回到原始来源。
- 人物观点是否明确归属，没有写成客观事实。
- 最新状态、数字、日期、职位和产品规则是否重新核对。
- 作者推论是否能指出依据。
- 用户经历、对话、情绪、耗时、成本和结果是否来自真实材料。
- 反例和不确定性是否被诚实保留。

任一核心事实无法核对，就降低语气、删除或暂停交付。

## 第二层 文章性

- 开头是否出现具体变化、人物、实践或冲突。
- 全文是否围绕一个核心问题推进。
- 每一段是否带来新事实、新机制、新反方或新判断。
- 段落长短是否有节奏，是否存在报告腔和提纲腔。
- 反方是否真实有力，而不是稻草人。
- 结尾是否完成判断升级，而不是总结目录。
- 结构是否符合所选原型。
- 面向公众号或 X 的 3000–8000 字长文，是否有 4–9 个能推进观点的二级标题；相邻阅读锚点是否大致间隔 400–700 字。
- 是否只加粗完整的关键判断、反转句或结论句，通常 6–16 处；是否避免只加粗零散名词。
- Markdown 段落之间是否只留一个空行，没有连续空行造成的空白段落。

标题和加粗是浏览导航，不是装饰。标题应该让只扫一遍的人也能看出论证如何推进；加粗句应该单独截出来仍然准确。观点型文章可以连续推进，但“连续”不等于三四千字没有阅读锚点。

## 第三层 情绪与传播

- 是否明确读者进入文章时的原有情绪。
- 开头约一百五十字内是否出现张力。
- 中段是否至少发生一次意外、反转或问题加深。
- 情绪是否有起伏，而不是全程一个音量。
- 是否有两到三句可独立传播且仍然准确的表达。
- 网感是否来自自然口语和当下语境，而不是热词拼贴。
- 结尾是否有情绪余波。

情绪不足时，回到矛盾和代价，不靠增加感叹号补救。传播句不足时，重新压缩核心判断，不单独制造金句。

## 第四层 个人语气

- 是否像一个做过事情、愿意承担判断的人。
- 是否平视读者，不装导师或品牌公关。
- 是否保留 Ray 的清楚、克制和证据意识。
- 是否有真实立场，而不只是把资料组织得很顺。
- 是否误用了别人的口头禅、人设和经历。
- 删除署名后，是否仍能看出独特判断。

## 机械检查后的人工复核

`article_check.py` 只能发现篇幅、格式、模板词、过度重复和部分节奏问题。它无法判断事实是否真的可靠，也无法证明一句话有传播力。

检查结果分两类：

- 错误：必须修改后才能交付。
- 提醒：必须逐项判断，确认合理才能保留。

最终交付前，从头朗读一遍。读不顺的地方直接改，不用“这是书面语”替它辩护。
