# Ray 封面视觉体系

这些是可迁移的编辑设计原则，不是对某个品牌或作者的风格复制。

## 1. editorial-metaphor-collage

适合身份、信任、平台、关系、权力和抽象机制。

- 高对比纯色或微纹理底色
- 黑白网点人物或物件剪影，叠加少量彩纸形状
- 3–6 个视觉元素，只组成一个关系
- 允许箭头、轨迹、门、秤、容器、锚点等关系符号
- 画面像一张有主张的杂志插图，不像素材拼盘
- 标题区保留 35%–45% 干净负空间

不要：品牌标志、密集小卡片、真实 UI、满屏标注、复杂信息图。

## 2. minimal-metaphor

适合一句判断足以撑住全文的观点文章。

- 70%–85% 留白
- 一个精确物体关系，最多一个小人物
- 奶油白、墨黑，加一个克制强调色
- 手工墨线、纸张纤维或轻微套印偏差
- 先问“删到只剩什么仍然成立”

不要：为了显得高级而留下无意义空白，或用抽象几何掩盖隐喻不清。

## 3. midcentury-editorial

适合方法论、产品判断、组织变化和有结构的观点。

- 中世纪现代主义书封构图
- 几何轮廓、平面色块、黑与奶油色，加一个锈红或钴蓝
- 大尺度形状形成秩序，小人物只负责比例或动作
- 画面稳、硬、有印刷物感

不要：做成怀旧海报复制品，或加入与主题无关的复古装饰。

## 4. quiet-ink

适合反噪声、创作、文字、真实性和慢变量。

- 固定暖米色纸底
- 黑色点阵、油墨或打字机式物体隐喻
- 少量机械排布与明显留白
- 情绪克制，像一张安静的旧印刷品

不要：脏旧滤镜过重、假做旧、咖啡渍和泛黄边框。

## 5. tool-bridge

适合教程、两个工具协作、输入输出和迁移关系。

- 两端各一个清楚对象，中间只有一条桥、线、管道或接头
- 关系优先于产品图标
- 若必须使用标志，只能来自用户提供的正式素材
- 背景简单，标题与对象互不竞争

不要：未经提供就凭记忆绘制商标，不做软件截图拼贴。

## 可选材料实验

- 银色薄片与纯蓝：适合硬科技、反射、边界和信号。
- 木屑压板与工业蓝：适合制造、搭建、粗粝过程和幕后系统。
- 撕纸微缩场景：适合旅程、地形、路线和有温度的叙事。

材料必须服务判断，只在用户明确选择时进入生产。

## 选择优先级

1. 隐喻是否一眼可懂。
2. 是否和文章核心判断一一对应。
3. 缩略图下是否仍然清楚。
4. 是否像 Ray 的编辑判断，而不是常见 AI 海报。
5. 风格新鲜度最后考虑。
