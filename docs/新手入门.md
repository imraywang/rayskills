# rayskills 新手入门

rayskills 是 Ray 在真实业务里沉淀的 AI Skill 工具箱。不是"教你怎么想"的诊断器,是**直接把下一步干掉**的 builder 工具——每个 skill 背后都是一段真实发生的实践。

## 怎么用

熟悉后直接调用具体 skill,命令后面跟材料或问题:

```text
/anvil 宠物陪伴 / 25-35岁独居女性 / 79元 / 一二线城市
/ray-thread 上周清了 8 台 VPS 的矿马,想发条 thread
/ray-vpsinit root@1.2.3.4
/ray-diagnose 一家做化妆品的公司想上 AI 客服,帮我评估能不能干
```

（`/ray` 主路由待成员定型后上线;届时可交任务自动分发。）

## Skill 全目录

### 基建

- **`/ray-vpsinit`**｜新 VPS 开荒。连通盘点 → 系统加固(SSH/BBR/更新/swap) → 代理栈(3x-ui+Reality) → 防火墙 → 验证交接。带防锁死保护和实测过的版本坑位。
- **`/ray-nodecheck`**｜节点健康巡检。逐节点探存活/延迟/出口 IP,读中转链探针日志,核订阅有效性,输出红黄绿健康表。只读不改。

### 内容

- **`/ray-thread`**｜实战经历 → build-in-public thread 装配骨架。素材考古、结构选型、逐条排布、hook 候选。只交骨架和素材,观点血肉作者自己填(不代笔)。
- **`/ray-tweet`**｜每日 𝕏 声部 C(社会观察/人生哲理)推文候选批次。
- **`/ray-report`**｜magazine 风格长文/深度报告,三格式输出(交互 HTML / A4 PDF / 公众号 markdown)。
- **`/ray-benchmark`**｜(排期)对标拆解:拆产品、拆定价、拆增长。

### 咨询

- **`/ray-diagnose`**｜(在建)企业知识库/AI 落地前置诊断。六维评估 → 病灶定位 → 红黄绿可行性评级。
- **`/ray-proposal`**｜(排期,私有)诊断结论 → 方案蓝图。

### 产品

- **`anvil`**｜铁砧。从消费社会批判框架锻造经得起拆的产品概念。
- **`/ray-launch`**｜(排期)落地页/B2B 站上线全流程。

## 设计原则

1. **有尸体或有账本**——每个 skill 背后要么是真实事故复盘,要么是真实收入业务
2. **一 skill 一现金流线**——喂 relay 引流 / 企业咨询 / IP 增长,不做纯炫技
3. **skill 即内容**——每个成熟 skill 配一条实战 thread,发布即营销
